include/include_once/require/require_once

  • php 파일을 로드하는 4가지 형식
  • incldue와 require의 차이는 존재하지 않는 파일의 로드를 시도했을때 include가 warning 을 일으킨다면 require는 fatal error를 일으킨다.
  • _once라는 접미사가 붙은 것은 파일을 로드 할때 단 한번만 로드 된다는 의미. 두번째는 포함하지 않는다.

네임스페이스

  • 파일안에서 이름을 붙여서 호출할수 있는 이름을 붙여주는 방식


greeting_en.php 

	function welcome(){
		return 'Hello world';
	}
greeting_ko.php 

	function welcome(){
		return '안녕하세요?';
	}
main.php 

		require_once 'greeting_ko.php';
		require_once 'greeting_en.php';
		echo welcome();
		echo welcome();
		
		php는 함수 중복선언을 허용하지 않는다.
		위와 같이 두파일에서 각각 welcome함수가 선언되엇기때문에
		오류가 발생한다. 이때 필요한것이 namespace이다.
greeting_en_ns.php 

	namespace language\en;
	function welcome(){
		return 'Hello world';
	}
greeting_ko_ns.php 

	namesapce language\ko;		
	function welcome(){
		return '안녕하세요?';
	}
main.php 

		require_once 'greeting_ko_ns.php';
		require_once 'greeting_en_ns.php';
		echo language\ko\welcome();
		echo language\en\welcome();
		
		로드되는 파일의 맨위에 namesapce를 이용해서 네임스페이스를 만듬. 
		네임스페이스를 사용할 때는 함수앞에 네임스페이스의 이름을 붙여서 사용하면된다.

	namesapce language\ko;

	}
	namespace language\en;

	}
main.php

		echo language\ko\welcome();
		echo language\en\welcome();
		
			하나의 파일에는 복수의 네임스페이스가 존재 할 수도 있다.
